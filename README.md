# AI Sales Agent

Dieses Projekt ist ein KI-gestütztes CRM- und Sales-Automation-System.

## Ziel
Automatisierte Lead-Recherche, Bewertung, Personalisierung und Verkaufsunterstützung mit KI-Agenten.

## Stack
- Python (FastAPI)
- React (Frontend)
- PostgreSQL
- Redis
- Docker

## Schnellstart
1. Repository klonen
	```bash
	git clone <repo-url>
	cd AI-Sales-Agent
	```
2. Environment einrichten
	```bash
	cp .env.example .env
	# .env anpassen
	```
3. Dienste starten (Docker)
	```bash
	docker-compose up --build
	```

## Hinweise
- API-Dokumentation: /docs (FastAPI)
- Datenbank-Migrationen mit Alembic
